# Just Life Clothing UX/UI Files

This set of files contains the UX/UI planning package for the Just Life Clothing website.

## Files

- `UX-UI-Brief.md` documents the audience, goals, experience principles, and page priorities.
- `UX-UI-User-Flows.md` maps the main shopping journeys from landing to cart.
- `UX-UI-Wireframes.md` outlines the page structure for desktop and mobile.
- `UX-UI-Design-System.md` defines the color, type, spacing, component, and interaction rules.
- `UX-UI-Prototype.html` is a browser-viewable visual design board using current site assets.
- `UX-UI-Prototype.css` styles the visual design board.

## Recommended Use

Open `UX-UI-Prototype.html` in a browser to review the visual direction, then use the markdown files as the written UX/UI documentation for the portfolio project.
